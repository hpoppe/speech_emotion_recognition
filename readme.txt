Overview of folders:
    Emo_DB: Experimental Setup for the Berlin Database of Emotional Speech (Emo-DB)
    TESS: Experimental Setup for the Toronto Emotional Speech Set (TESS)
    MSP_Podcast_Corpus: Experimental Setup for the MSP-Podcast corpus
    Open_Smile_Features: Different approach to the dataset creation for the Emo-DB based on config files by https://audeering.github.io/opensmile/index.html

You can find a readme text file in each of these folders.
